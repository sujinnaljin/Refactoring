## 스터디 진행 방법

- 1- 4장은 책에서 꼼꼼히 보는걸 추천. 따라서 아래 내용들을 정리
  - 중요 내용
  - 개인적인 느낌, 생각
  - 서로 논의해볼 내용
  - 추가로 1, 4장에 나오는 예제 코드는 자신이 사용하는 언어로 실습

- 카탈로그 장 (6 - 12 장) 도 위와 똑같은 방식으로 진행 하다가 더 좋은 방식이 떠오르면 다시 얘기
